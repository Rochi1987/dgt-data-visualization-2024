# dgt-data-visualization-2024
Código en Python para la extracción, depuración y visualización de la información general de datos municipales de tráfico en España (2024). Analiza la relación de conductores por género, volumen de turismos por región y la composición ambiental del parque automotor a través de gráficos estadísticos y mapas interactivos basados en GeoJSON.
